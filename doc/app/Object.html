generate_l10n_file() click to toggle source

Generates a best-estimate l10n file from all views by collecting calls to _() — note: use the generated file only as a start (this method is only guesstimating) 

     # File lib/localization.rb, line 95
 95: def self.generate_l10n_file
 96:   "Localization.define('en_US') do |l|\n" <<
 97:   Dir.glob("#{Rails.root}/app/views/**/*.rhtml").collect do |f|
 98:     ["# #{f}"] << File.read(f).scan(/<%.*[^\w]_\s*[\"\'](.*?)[\"\']/)
 99:   end.uniq.flatten.collect do |g|
100:     g.starts_with?('#') ? "\n  #{g}" : "  l.store '#{g}', '#{g}'"
101:   end.uniq.join("\n") << "\nend"
102: end
