add_foreign_keys_for(tables) click to toggle source

Adds foreign keys for all tables given. Tables should be hash where the target table name is the key. The key should point to an array of all table names which should reference that target. 

 
    # File lib/migration_helpers.rb, line 26
26:   def add_foreign_keys_for(tables)
27:     tables.each do |reference_name, tables|
28:       column_name = reference_name.to_s.singularize.foreign_key
29: 
30:       tables.each do |table| 
31:         begin
32:           foreign_key(table, column_name, reference_name)
33:         rescue
34:           puts "ERROR"
35:           puts $!
36:         end
37:       end
38:     end
39:   end

 6:     constraint_name = "fk_#{from_table}_#{from_column}" 
 7: 
 8:     cmd = "alter table #{from_table} "
