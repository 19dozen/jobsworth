filter(objects, filter_params = {}) click to toggle source

Runs through the given objects and returns only those which return one of the given values for each method given. The methods must be declared in the class in a class variable called FILTERABLE.

For example, to only include objects that return “a”, or “b” from a method called “name”, the filter_params should be: { :name => [ “a”, “b” ] }.

Any filters based on methods not in FILTERABLE are ignored.
